# 3D Cube Game

An interactive Rubik's Cube puzzle game built with Three.js and Vite.

**Live at:** https://solvx.uk/cube-game/

## Features
- Full 3D Rubik's Cube with face rotation
- Click + drag to rotate faces
- Keyboard shortcuts (R, L, U, D, F, B + Shift for reverse)
- Scramble and Reset buttons

## Planned Features
- Timer and move counter
- Solve detection
- Mobile touch controls
- Colour themes
- Leaderboard

## Getting Started

```bash
git clone https://github.com/tradewithmeai/3d-cube-game.git
cd 3d-cube-game
npm install
npm run dev
```
