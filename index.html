Rubik's Cube

Controls:

Click + drag on cube to rotate faces

Drag background to rotate view

Keys: R, L, U, D, F, B (+ Shift for reverse)

About This Project

This Rubik's Cube game is a live development experiment hosted on solvX.uk. It's built with Three.js and deployed automatically on every code change.

Why is it here?

We believe the best way to learn is by building in public. This project is intentionally unfinished — it's a playground for developers to experiment, contribute, and learn.

Known Bug: There's a bug with how sections of the cube rotate during gameplay. The game runs and is playable, but something isn't right. Can you spot it? Can you fix it? This is a real open issue — check it on GitHub.

What's next?

  • Fix the rotation bug (see above!)
  • Timer and move counter
  • Solve detection and win state
  • Mobile touch controls
  • Colour themes and customisation
  • Leaderboard (coming later)
